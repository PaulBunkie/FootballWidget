# BetON — Железобетонные ставки

Не следите за матчами — BetON сделает это за вас.

Приложение автоматически отслеживает футбольные матчи с явным фаворитом и сообщает только тогда, когда происходит действительно важное событие.

## Главная особенность — «Фаворит в беде»

В течение дня проходят сотни футбольных матчей, и следить за всеми невозможно. BetON делает это за вас.

Приложение автоматически отслеживает матчи, в которых есть явный фаворит. Пока всё идёт по ожидаемому сценарию, оно не отвлекает. Но если фаворит неожиданно начинает проигрывать или теряет преимущество, BetON мгновенно отправляет уведомление **«Фаворит в беде»**.

Откройте матч одним касанием, оцените, что происходит на поле по графику атак и другим live-показателям, и самостоятельно решите, заслуживает ли эта ситуация вашего внимания.

BetON не даёт прогнозов и не подсказывает, как поступить. Он просто помогает не пропустить момент, когда привычный ход игры меняется.

## Возможности

⚽ **Live-виджет** с текущими и предстоящими матчами прямо на главном экране.

⭐ **Настройка силы фаворита** — выберите, насколько явным должно быть преимущество команды, чтобы матч попал в список отслеживания.

🔔 **Мгновенные уведомления**

* начало матча;
* забитые голы;
* **«Фаворит в беде»** — когда фаворит начинает проигрывать или теряет преимущество;
* окончание матча.

📊 **Быстрый переход к матчу** — одним нажатием откройте подробную информацию, график атак и другие live-показатели, чтобы оценить происходящее.

⚡ **Мгновенные обновления** через Firebase Cloud Messaging без постоянного опроса серверов.

🔋 **Экономное энергопотребление** — приложение не расходует заряд батареи лишними фоновыми запросами.

🌍 **Поддержка нескольких языков**: русский, English, Українська, Беларуская, Қазақша, Español, Français, Deutsch и 中文.

BetON не пытается заменить спортивный портал или приложение букмекерской конторы. Он берёт на себя самую скучную работу — следит за матчами и сообщает только тогда, когда действительно происходит что-то важное.
